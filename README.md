**CRM Python pour Go High Level**
Ce projet est une solution CRM développée en Python, conçue pour faciliter la gestion et l’importation des données clients dans Go High Level. Il permet de faire le mapping automatique des champs CSV et de diviser de gros fichiers CSV en plusieurs parties pour simplifier l’importation dans la plateforme.

**Fonctionnalités principales**
- Mapping des colonnes CSV pour Go High Level (utilisation de pandas).
- Division automatique des fichiers CSV volumineux en fichiers plus petits.
- Nettoyage et organisation des données clients avec Python et pandas.
